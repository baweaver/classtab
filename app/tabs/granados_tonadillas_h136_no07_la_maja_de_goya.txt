LA MAJA DE GOYA a Tonadilla by Enrique Granados

Transcribed by David Kahts

This is one of the most poetical pieces I have ever come across as a classical
guitarist. It is very difficult in some places and needs a lot of dedication
if you want to get it as flowing and graceful as Andres Segovia's or Julian
Bream's interpretation. - dkahts@tsamail.trsa.ac.za [no longer active]

The_S: I've come a long way since my last effort to tab this piece.
       I now recognise the need for bars and more importantly,
       note durations...

       (example tab)

         |   |   |    |    |      |    |   |    |    |    |     | 
       |---------3--|-10---8h10p8----|----------3--|-8----6h8p6----|
       |------------|-------------12-|-------------|------------10-|
       |-----3------|------10=====---|-----3-------|------8=====---|
       |-.----------|----------------|-------------|---------------|
       |-0----------|-0====----------|-.-----------|-10===---------|
       |------------|----------------|-0-----------|---------------|
         ^            ^    ^               ^    ^
        (a)          (b1) (b2)            (c1) (c2)

(a) A dot either above or below a note represents pizzicato, which
    is a sudden and short-lived note.
(b) Notes which have been extended using '==' signs are to be played
    for as long as the '==' signs show. eg Note (b1) should last only
    until note (b2) and then be muted. Note (b2) should take over from
    note (b1).
(c) Notes which have NOT been extended using '==' signs should only
    last until the next note is played. eg Note (c1) should only last until
    note (c2). I did this to save time. Also, the tab would look messy if
    it were filled with '==' signs.

Note durations can mean the difference between a clumsy player (myself)
and a virtuoso. They can also help to improve your technique. If you
can sustain the notes in this piece for their proper durations, you will
have mastered the piece, and what a beautiful piece to master!

My apologies to Weed for failing to see the relevance of bars and note
durations in past efforts. Maybe one day I'll even make a tab with fingering!
