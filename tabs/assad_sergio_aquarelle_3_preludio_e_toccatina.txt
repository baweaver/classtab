Performance Notes
-----------------

* Measure 2: The sheet music denotes the Bb in the first beat as a crotchet (quarter
             note) to be held by finger 4. There is no practical way to do this. An
             ossia can be found below with different fingering.
                |       |       |       |
            e|--0===============================|
            B|----------1---0-------------------|
            G|--3=======------2-0h2---0-----0-2-|
            D|------7-5---0---------3---2h3-----|
            A|----7-------------3=======1===0---|
            D|--0-----------------------------5-|

* Measure 3: Three against four is always difficult to properly represent in tab. Below is how the
             measure would look if the middle voice were separated. Note the triplets are only in
             the middle voice.
                |       |       |       |          |         |       |         |
            e|--0=====0=====0=====0=====1=======|  ....3....         ....3.... ....3... 
            B|                                  |--------------------------0------------|
            G|                                  |-----2----------7---5--5------2--0-----|
            D|                                  |--2-----5---3-----------------------3--|
            A|------------------0=======1=======|
            E|--3=======5=======----------------|

* Measure 9: In the second beat, the sheet music denotes the lower Bb as a crotchet.
             It is impossible to play as written.

* Measure 31: The sheet music recommends playing the Bb in the first beat on the A string,
              but this is nigh impossible while holding C on the G string for the specified
              duration.

===    held note
p      pull-off
h      hammer-on
/      slide up
\      slide down
(      arpeggiate - slowly strum from the lowest string to the highest string
)
>      marcato (stress the note or chord)
^      sforzando (heavily stress the note or chord)
-      tenuto (hold the note for its full duration)
.      staccato (shorten the note duration)
:      mezzo staccato (play detached)

/.\    fermata
- >    crescendo (getting louder)
< -    decrescendo (getting softer)
cresc. crescendo (getting louder)
ff     fortissimo (very loud)
f      forte (loud)
mf     mezzo forte (medium loud)
mp     mezzo piano (medium soft)
p      piano (soft)
pp     pianissimo (very soft)
sub.   subito (suddenly)

métallique, metalico     metallic (play near the bridge)
lent et très expressif   slow and very expressive

<5>    natural harmonic - play open string, touch point on 5th fret
<7>    natural harmonic - play open string, touch point on 7th fret
<9>    natural harmonic - play open string, touch point on 9th fret
<12>   natural harmonic - play open string, touch point on 12th fret
[5]    artificial harmonic - play indicated fret, touch point on 17th fret
