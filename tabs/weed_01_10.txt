an easy piece, to be played solemnly, with dignity

bar  2 : slide up from held E in bar 1
bar  4 : only need to barre strings 3,4
bar 40 : barre strings 1,2,3
bar 42 : barre strings 2,3 - then barre strings 1,2
bar 44 : only need to bar strings 1,2
bar 46 : only need to bar strings 1,2
bar 50 : only need to bar strings 2,3

CII  : barre 2nd fret (similarly CI, CIII, CV)
 =   : note is held
 /   : slide up
 *   : start/end of repeated section
ferm : fermata (pause)

fingering - 1  1st (fore) finger
            2  2nd (index) finger
            3  3rd (ring) finger
            4  4th (little) finger (pinkie)
